SheetClaw Privacy Policy

Last updated: June 2026

SheetClaw is a personal-use Microsoft Excel task pane add-in. This policy describes what data the add-in collects, how it is stored, and where it is sent.

Data stored on your device

All data SheetClaw stores is kept locally in your browser's localStorage, scoped to the add-in's origin. It is never uploaded to any server operated by SheetClaw. The following is stored:

  • LLM provider credentials — API keys and OAuth tokens you enter in Settings. These are stored so you do not need to re-enter them each session.
  • Provider configuration — your chosen provider, base URL, and model name.
  • Usage history — estimated token counts and costs derived from your chat sessions, stored as daily buckets. No message content is retained.
  • Session state — the current chat session's message history, kept only for the lifetime of the task pane.

You can clear all stored data at any time by removing the add-in from Excel or clearing your browser's localStorage for the add-in's origin.

Data sent to third-party LLM providers

When you send a message in the chat interface, SheetClaw forwards your prompt and relevant workbook context (cell values, sheet names, named ranges) to the LLM provider you have configured. This may include:

  • Text you type in the chat input.
  • Cell contents from the active workbook that the agent reads as context.
  • Workbook metadata such as sheet names and named range names.

SheetClaw sends this data directly from your device to the provider's API using your own credentials. The provider's own privacy policy governs how that data is handled. SheetClaw does not act as an intermediary and retains no copy of this data after the request completes.

Supported providers include Ollama (local, no data leaves your device), OpenAI, Anthropic, OpenRouter, DeepSeek, Groq, Mistral, Together AI, Kimi, GLM, Qwen, Llama, and any compatible endpoint you configure.

Data not collected

SheetClaw does not collect:

  • Analytics or telemetry of any kind.
  • Crash reports or usage statistics sent to a SheetClaw server.
  • Personal information such as your name, email address, or location.

Children

SheetClaw is not directed at children under 13 and does not knowingly collect data from them.

Changes to this policy

If this policy changes, the updated version will be published at this URL with a revised date. Continued use of the add-in after a change constitutes acceptance of the updated policy.
